# School Donation Management System

This is a Java-based web application developed to manage donations for schools.

## Features
- User registration and login
- Donation submission and tracking
- Admin dashboard for managing donations
- School request management

## Technologies Used
- Java
- JSP & Servlets
- Apache Tomcat
- MySQL

## How to Run
1. Import the project into Eclipse
2. Configure Apache Tomcat server
3. Import the database using database.sql
4. Run the project on the server
